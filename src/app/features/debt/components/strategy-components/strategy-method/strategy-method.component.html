@if (metodo() === 'bola-de-nieve') {

El método Bola de Nieve se enfoca en ganar impulso psicológico al eliminar primero las deudas más pequeñas. Sigue estos pasos:

  1. Ordena tus deudas de menor a mayor por cantidad total.
  2. Destina el dinero extra a pagar la deuda más pequeña primero.
  3. Una vez pagada, suma ese pago a la siguiente deuda más pequeña.
  4. Continúa el proceso hasta eliminar todas tus deudas.
} @else {

El método Avalancha se enfoca en minimizar los intereses pagados al atacar primero las deudas más costosas. Sigue estos pasos:

  1. Ordena tus deudas de mayor a menor por tasa de interés.
  2. Destina el dinero extra a pagar la deuda con mayor interés primero.
  3. Una vez pagada, suma ese pago a la siguiente deuda con mayor interés.
  4. Continúa el proceso hasta eliminar todas tus deudas.
}

Recordatorio importante

Siempre continúa pagando el mínimo en todas tus deudas mientras aplicas el método.
